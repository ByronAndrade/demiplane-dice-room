# Demiplane Dice Room Support

For support, bug reports, and feature requests, use the GitHub issue tracker:

https://github.com/ByronAndrade/demiplane-dice-room/issues

When reporting a bug, include:

- Browser and version.
- Extension version.
- Whether you were using the community relay, a local relay, or a self-hosted relay.
- Whether you were host/storyteller or player.
- What action caused the issue.
- Any visible diagnostic message from the Dice Room panel.
